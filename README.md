# KingOfTime Input Subitem Extension

KingOfTime 補助項目入力を効率化するための拡張機能です。
使い方は画像で説明しているので見てください。

### 機能
以下の補助項目入力を効率化します
- あらかじめジョブリストの中からプリセットを登録し、プルダウンの下にボタンを自動配置することで簡単にジョブを選択できます
- 打刻時間を認識し、自動的に時間を入力するようなボタンを配置します
